# EventFlow - Microservices Order Platform

A real-time event-driven microservices system built using FastAPI, Kafka, Redis, MongoDB, and React.

## Features
- User authentication & authorization (RBAC)
- Item & inventory management (Admin)
- Order processing system
- Real-time updates using WebSockets
- Event-driven architecture using Kafka
- Redis caching (DB + Query level)
- API Gateway with NGINX
- Dockerized services
- CI/CD with GitHub Actions
